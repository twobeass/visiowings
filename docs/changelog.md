# Changelog

The canonical changelog lives in
[`CHANGELOG.md`](https://github.com/twobeass/visiowings/blob/main/CHANGELOG.md)
at the repository root and is generated automatically by
[release-please](https://github.com/googleapis/release-please) from
Conventional Commit messages.

You can also browse releases (with attached `.exe`, SBOM, and license
report) on the
[GitHub Releases page](https://github.com/twobeass/visiowings/releases).
